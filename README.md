# eslint-config-pfm

## Usage

An ESLint Configuration for PFM

### eslint-config-pfm

1. Install packages

```bash

npm info "eslint-config-pfm@latest" peerDependencies

# npm > 5.x+
npx instll-peerdeps --dev eslint-config-pfm

```

2. Update `.eslintrc` or `.eslintrc.js`

```json
extends: [
  "pfm",
],
```
